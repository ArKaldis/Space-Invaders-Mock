# Star Siege

Star Siege is a small browser arcade game that blends two classic ideas:

- Space Invaders style enemy waves
- Asteroids style drifting hazards

You pilot a small ship, clear invader formations, dodge falling rocks, and try to survive as long as possible while your score climbs.

## Features

- Canvas-based arcade gameplay
- Keyboard controls
- Score, lives, and wave tracking
- Enemy projectiles and asteroid hazards
- Restart button for quick replays
- Simple single-page setup with no build step

## How to Run

This project does not require installation.

1. Clone or download the repository.
2. Open `index.html` in a web browser.

On Windows PowerShell, you can also run:

```powershell
start .\index.html
```

## Controls

- `A` / `D` to move
- `Left Arrow` / `Right Arrow` to move
- `Space` to shoot
- `Shift` to dash
